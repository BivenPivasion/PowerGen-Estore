# ⚡️ POWERGEN — Energy Solutions E-store

**POWERGEN** — це сучасна платформа для продажу промислового та побутового енергетичного обладнання (генераторів, портативних зарядних станцій, систем безперебійного живлення). Проєкт розроблений з акцентом на архітектурну чистоту, безпеку та готовність до деплою.

---

## 🛠 Технологічний стек (Tech Stack)

* **Backend:** Python 3.12, Django 5.x
* **Database:** PostgreSQL (реляційна БД для надійного зберігання даних)
* **DevOps:** Docker & Docker Compose (повна контейнеризація проєкту)
* **Static Serving:** WhiteNoise (оптимізована роздача статики)
* **Frontend:** HTML5, CSS3 (Custom Design), JavaScript, FontAwesome

---

## 🌟 Ключовий функціонал (Key Features)

* **Повноцінний каталог товарів:** Зручна навігація за категоріями.
* **Система пошуку та фільтрації:** Швидкий пошук за назвою та характеристиками.
* **Кошик та замовлення:** Повна логіка додавання товарів та оформлення замовлення.
* **Інтеграція оплати:** Підключений шлюз LiqPay (Sandbox mode).
* **Кабінет користувача:** Система реєстрації, авторизації та керування профілем.
* **Адмін-панель:** Професійний інтерфейс для керування асортиментом та клієнтами.

## 🚀 Як запустити проєкт локально

Для запуску вам знадобиться встановлений **Docker** та **Docker Compose**.

1.  Клонуйте репозиторій:
    ```bash
    git clone [https://github.com/BivenPivasion/PowerGen-Estore.git](https://github.com/BivenPivasion/PowerGen-Estore.git)
    cd PowerGen-Estore
    ```
2.  Створіть файл `.env` на основі `.env.example` та заповніть свої дані.
3.  Запустіть контейнери:
    ```bash
    docker-compose up --build
    ```
4.  Виконайте міграції:
    ```bash
    docker-compose exec web python manage.py migrate
    ```
5.  Сайт буде доступний за адресою: `http://127.0.0.1:8000`
